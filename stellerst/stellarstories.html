StellarStories

This project is a dynamic blogging website developed with Flask, HTML, CSS, and JavaScript. Users can register and log in to the site to read and contribute blogs.

Type:
  • Blogging Website
Frontend:
  • HTML
  • CSS
  • JavaScript
Backend

  • Flask
Link:
StellarStories

The homepage displays recent blogs with essential details. Clicking on a blog card reveals a detailed view, including full content, author information, and comments. Authenticated users can create, edit, and delete their blogs. The technology stack comprises HTML for markup, CSS for styling, JavaScript for interactivity, and Flask for backend functionality, ensuring a secure and user-friendly blogging platform.
